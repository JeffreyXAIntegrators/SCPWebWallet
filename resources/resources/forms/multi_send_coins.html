Send ScPrime coins and funds to multiple addresses in the same transaction as defined in the supplied CSV file. The first column must be the amount of ScPrime coins or funds to send specified in units, e.g. 1.23KS or 30SPF. The second column must be the 76-byte hexadecimal address. 
    Units:
The smallest unit of scprimecoins is the Hasting.
One scprimecoin is 10^27 Hastings.
Other supported units are:
  pS (pico,  10^-12 SCP)
  nS (nano,  10^-9 SCP)
  uS (micro, 10^-6 SCP)
  mS (milli, 10^-3 SCP)
  SCP
  KS (kilo, 10^3 SCP)
  MS (mega, 10^6 SCP)
  GS (giga, 10^9 SCP)
  TS (tera, 10^12 SCP)

  SPF-A and SPF-B do not support fractional units.
  To send SPF-A use "SPF" or "SPF-A" unit suffix.
  To send SPF-B use "SPF-B" unit suffix.
A dynamic transaction fee is applied depending on the size of the transaction and how busy the network is.
