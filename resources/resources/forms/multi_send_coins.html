Send ScPrime coins and funds to multiple addresses in the same transaction as defined in the supplied CSV file. The first column must be the amount of ScPrime coins or funds to send specified in units, e.g. 1230SCP, 30SPF, 30SPF-A or 10SPF-B. If no units are specified, default unit type will be used. The second column must be the 76-byte hexadecimal address.

SPF-A and SPF-B do not support fractional units. To send SPF-A use "SPF" or "SPF-A" unit suffix. To send SPF-B use "SPF-B" unit suffix.

A dynamic transaction fee is applied depending on the size of the transaction and how busy the network is.
